# status monitor

A UI to visualize the data.<br/>
Clean architecture is implemented!

<hr/>

## Get started

1. install
```bash
$ yarn install
```

2. copy .env
```bash
$ cp .env.template .env
```

3. copy setting.json
```bash
$ cp ./public/setting.json.template ./public/setting.json
```

you can controll the UI by updating .env and setting.json, have fun!
